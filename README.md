# skill-flyctl

A [skill](https://skills.sh) for deploying and managing applications on Fly.io using the flyctl CLI.

## What is this?

This skill provides AI agents with knowledge about Fly.io deployment workflows, including app launch, deployment, scaling, secrets management, volumes, and databases.

## Installation

```bash
npx skills add nickspaargaren/skill-flyctl
```

## Usage

Once installed, your AI agent will automatically use this skill when you mention:
- `fly deploy`
- `fly.io`
- `flyctl`
- Deploying to Fly

The skill provides guidance on:
- Launching new apps (`fly launch`)
- Deploying applications (`fly deploy`)
- Managing secrets and environment variables
- Scaling instances and resources
- Working with volumes and databases
- Monitoring and troubleshooting
